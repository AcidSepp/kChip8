# kChip8

This is a WiP in progress emulator for Chip8 written in Kotlin.
This is an educational project highlighting some important aspects of computer architecture:

* machine code
* memory organisation
* registers
* low level binary operations
* ...

I also try to emphasize the indispensability of fine-grained unit testing.
